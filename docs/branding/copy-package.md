## ⑧ 사용 가이드

**일관성 우선** — 어디서든 핵심 진단·핵심 구절은 동일:
> **글로컬 시대는 이주민 시대.**
> **이주민과의 상생이 한국 사회와 교회를 다시 세웁니다.**

**문장 끝 어미** — '습니다' 통일. 시대 진단의 무게에 어울리는 진중함.

**금지 표현**:
- "30년" 등 시간 누적 표현 (시간이 아닌 시대가 권위)
- "최고의, 완벽한, 유일한" (오버 셀링)
- "솔루션, 생태계, 플랫폼화" (비즈니스 클리셰)
- "도움을 드리는, 돕는, 지원하는" (시혜적 뉘앙스 — 상생 정체성과 모순)
- "다양한 분들과 함께하는" (모호함)
- "어려운 이주민" (대상화)

**선호 표현**:
- "이주민과 함께" (수평적 관계)
- "한국 사회와 교회를 다시 세우다" (상호적 효과)
- "동역자" / "시대의 응답" (사명 인식)
- "옆에 앉다", "눈을 마주치다" (구체적 행위)
- "거제·김해", "10여 개국" (구체적 장소·범위)
- "잇다", "만나다", "세우다" (브릿지·재건 정체성)

**숫자는 정확하게**:
- "10여 개국" — 사역 대상 국적 범위
- "250만 명 이상" — 한국 거주 이주민 (2024년 법무부 기준 약 250만)
- "9개 언어" — 한진준 목사 설교 (한·영·베·인니·태·신할라·러·네팔·캄)
- "5대 대상" — 근로자·유학생·청소년·다문화가정·해외유학준비생

**프레이밍 원칙**:
- 과거 (시간 누적) ❌ → 시대 (현재 부르심) ✅
- 권위 (이력) ❌ → 응답 (시대적 사명) ✅
- 사역자 중심 ❌ → 이주민·한국 사회 상호 갱신 ✅
- 시혜 모델 ❌ → 상생·동역자 모델 ✅
